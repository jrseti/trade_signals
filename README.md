# Experiments with Stock Market Trading Using LLMs

I am experimenting with using LLMs to see if pattern matching may be useful to generate signals for trading. Worth an experiment to me.

The patterns the experiments will look are very simple patterns.

## data
This directory contains training data culled from various source. The datetimes are in Pacific time. The data is culled to the New york session, 6:30an -> 1:15pm Pacific.

## ranges

An experiment detecting the opening range and looking for signals related to movement around this opening range.
